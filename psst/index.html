Features

Real-time scanning

Every message, every channel. psst watches in real time so secrets get caught in seconds, not days. No batch processing, no delays.

22+ built-in patterns

AWS, GitHub, Slack, Stripe, OpenAI, Anthropic, Telegram, SSNs, credit cards, private keys, JWTs, and more. Ready out of the box.

Self-hosted

Runs in your infrastructure via Docker. Your messages never leave your network. No SaaS middleman touching your Slack data.

Configurable actions

Thread reply, DM the user, alert a channel, auto-delete, or just log it. Set different actions per severity. Your rules, your workflow.

Audit trail

Every detection logged to SQLite with full context. Who posted what, when, which channel, what action was taken. Compliance-ready.

Friendly alerts

No shame, no blame. psst gives people a quiet heads up and helps them fix it. Security culture, not security theater.

How it works

01

Install

Create the Slack app from our manifest and run the Docker container. Five minutes, tops.

docker run -d gloom/psst:latest
02

Configure

Set your alert channels, tune pattern sensitivity, and decide what actions to take per severity. All via YAML or the Slack UI.

/psst config --channel #security-alerts
03

Relax

psst watches for secrets 24/7. When something slips, it handles it quietly. You get the audit trail, your team gets the heads up.
