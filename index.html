How it works

01

Install

Add psst to your Slack workspace. Takes about 2 minutes. We handle the infrastructure.

02

Configure

Define your rules in YAML or use the GUI. Choose what to detect, how to respond, where to alert. Config as code, version controlled.

03

Relax

psst watches every message in real time. When a secret slips, it handles it quietly. Your security team gets visibility, your engineers get a friendly heads up.

Config as code

Define your detection rules however you like. YAML for version control, or the GUI for quick edits. They stay in sync.

YAML GUI 
# psst.yaml
patterns:
  builtin:
    - aws
    - github
    - stripe
    - openai

  custom:
    - name: Internal API Key
      regex: acme_sk_[a-zA-Z0-9]{32}
      severity: critical

actions:
  critical: [delete, dm, alert]
  high: [reply, dm, alert]
  medium: [reply, alert]
  low: [log]

alert_channel: #security-alerts
exclude_channels: [#random, #social]

Features

Real-time detection
