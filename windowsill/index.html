TUI

Live dashboard in your terminal. Run sill and leave it in a tab. Keyboard shortcuts for everything.

CLI

Query commands for scripts and one-off checks. Pipe to jq, grep, whatever. sill connections --json

Menu Bar

Optional status icon. Green means normal. Yellow means new activity. Red means something flagged.

Commands

$ sill # open TUI dashboard
$ sill connections --last 1h # recent connections
$ sill ssh # active SSH sessions
$ sill git # git push/pull/fetch activity
$ sill listeners # open ports
$ sill baseline # manage learned baseline
$ sill export --json # dump for analysis

What it watches

Outbound connections

Every process that talks to the network. Domain, IP, port, timestamp. Know what's reaching out.

SSH sessions

Where you're connected, how long you've been there. Production access at a glance.

Git activity

Pushes, pulls, fetches. Which repos, which remotes. Track what code is moving.

Listening ports

What's exposed on your machine. New listeners get flagged. No surprises.

Baseline drift

Learns what's normal. Flags what's new. First time you hit a domain? You'll know.

All local

SQLite on your machine. No cloud. No telemetry. Export when you want, to where you want.
